IQRF SPI library for Arduino

![Build Status](https://travis-ci.org/iqrfsdk/clibiqrf-arduino.svg?branch=master) [

](LICENSE)

Supported IQRF TR modules

  • IQRF TR-72
  • IQRF TR-52

Supported platforms

This library is supported on AVR-based Arduino compatable platforms (e.g. Arduino Uno, Leonardo, Mega, Nano).

A library for Chipkit is there.

Installation

The best way how to install this library is to download a latest package or use a platformio:

``` platformio lib install 318 ```
